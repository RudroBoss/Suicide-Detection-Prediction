# Suicide-Detection-Prediction
Advanced suicide risk detection using CNN, BiLSTM, CNN-LSTM, LightGBM &amp; XGBoost with weighted ensemble. TF-IDF + handcrafted text features + deep embeddings. Includes attention visualization, LIME explanations, ROC/CM analysis &amp; full evaluation pipeline. Achieved 95.3% test accuracy (AUC≈0.989). TensorFlow 2.18 compatible.
